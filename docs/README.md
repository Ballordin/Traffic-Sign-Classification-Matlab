# Technical Documentation

This folder contains the formal engineering report detailing the computer vision methodology.

## Key Chapters:
* **Pre-processing:** Techniques used to prepare the raw webcam feed for analysis.
* **Segmentation Strategy:** Detailed explanation of the HSV color space choices and dynamic thresholding.
* **Classification Algorithm:** The decision tree logic used to distinguish between different traffic signs.
* **Metric Validation:** Analysis of the system's performance using Confusion Matrices and error logs.
